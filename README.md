# Spotify Clone (HTML & CSS)

This is a simple Spotify clone UI built using only HTML and CSS.
The project focuses on recreating the look and feel of Spotifyâ€™s
web interface for learning and practice purposes.

## Overview
This project was created to improve my frontend development skills,
especially HTML structure and CSS styling. It is a static UI project
and does not include backend or music streaming functionality.

## Features
- Spotify-like user interface
- Clean and responsive layout
- Sidebar with navigation menu
- Music cards and playlist section
- Footer music player design (UI only)

## Technologies Used
- HTML
- CSS

## How to Run the Project
1. Clone or download the repository
2. Open the project folder
3. Open `index.html` in your browser
4. Explore the Spotify-style interface
